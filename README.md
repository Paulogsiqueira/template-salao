# 💇 Studio Bella — Template para salões de beleza e estética

Landing page responsiva como modelo de catálogo, parte do meu portfólio.

**Demo ao vivo:** _(GitHub Pages)_

## Características
- Página única responsiva (mobile-first), sem dependências ou build
- HTML5 + CSS3 + JavaScript puro
- Visual elegante (rosé e dourado), sob medida para o nicho
- Menu mobile, rolagem suave e formulário de contato (demonstração)

## Como usar
Abra o `index.html` no navegador — não há instalação. Para customizar, edite os textos no `index.html`, as cores em `:root` no `styles.css` e troque as imagens (URLs do Unsplash).

## Estrutura
```
index.html   · marcação e conteúdo
styles.css   · estilos e variáveis de tema
script.js    · menu mobile e ano do rodapé
```
